# Vibethone

🧠 AIML Lab – Interactive AI Learning Platform

AIML Lab is an interactive web platform designed to help beginners learn Artificial Intelligence & Machine Learning concepts through gamified modules, quizzes, mini-games, and simulations.

This project focuses on learning by doing instead of passive reading.

🚀 Live Features
📚 Learning Modules

Structured AI/ML topics from beginner to advanced:

What is Machine Learning
Supervised vs Unsupervised Learning
Decision Trees
Neural Networks
Transformers & Attention

Each module includes:

Simple explanations
Real-world examples
Visual learning aids
XP reward system
✏️ Quiz System

Interactive quiz to test understanding:

Multiple choice questions
Instant feedback
Score tracking
XP rewards
Badge unlocking system
🎮 Mini-Game Hub

Gamified learning experience:

Algorithm selection challenges
Speed classifier game
Streak & reward mechanics
🔬 AI Simulation Mode

Hands-on intuition building through interactive simulations.

📊 Dashboard

Tracks learner progress:

Modules completed
XP collected
Achievements & badges
Leaderboard
🏆 Gamification System
Feature	Description
⚡ XP Points	Earned by completing modules & quizzes
🏅 Badges	Rewards for achievements
🔥 Streaks	Encourages consistent learning
🥇 Leaderboard	Competitive motivation
